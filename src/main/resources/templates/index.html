Gestión integral para pisos compartidos

Aplicación orientada a personas que comparten piso y necesitan coordinar de forma clara y justa el dinero, la organización doméstica y el tiempo.

El sistema se estructura en cinco módulos principales, cada uno con funcionalidades específicas pero conectadas entre sí para dar una visión global del estado del piso.

Módulos

  • Gastos compartidos: control de gastos, reparto y liquidaciones. ver
  • Compra del super: organización de compras y reparto flexible. ver
  • Home: resumen del estado del piso y eventos. ver
  • Tareas: gestión de tareas domésticas y rotaciones. ver
  • Calendario: planificación y disponibilidad compartida. ver
